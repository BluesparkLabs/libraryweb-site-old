To add an icon:

1. Go to http://icomoon.io/app/
2. Upload SVGs for *all* icons
3. Download font
4. remove current icon dir, replace with downloaded font dir
5. Fix permissions (chmod a+rx icons)
6. Edit downloaded json + css to replace underscores with dashes. (see git history)
6. edit uclalib_arch.module and add any new icons.
